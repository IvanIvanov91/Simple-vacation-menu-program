A very simple program written in C++ for vacations using the folowing criteria:

Програма тип „меню”  за поддържане на информация за екскурзии със следните изисквания:
1.	Данните за екскурзиите се съхраняват в двоични файлове. За всяка екскурзия се пазят следните данни:
•	Уникален код (4 символа)
•	Дата на заминаване (проверка за допустимост на датата)  -  записана във формата ДД.ММ.ГГГГ;
•	Продължителност в дни -  цяло число;
•	Цена  -  реално число;
2.	Да се извършват следните обработки: 
а)  добавяне на нова екскурзия;
б)  извеждане на данните за най-дългата екскурзия;
в)  извеждане информацията за всички екскурзии с изтекла дата на заминаване (текущата дата се задава от потребителя);
г)  изтриване на данните за екскурзия с посочен код.
Данните да се поддържат в динамична структура - едносвързан списък в оперативната памет на ПК.
